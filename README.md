# World Explorer 🌍

Aplicación profesional construida con **React + TypeScript + TailwindCSS** que permite explorar información de todos los países del mundo usando la API de [REST Countries](https://restcountries.com/).

## ✨ Funcionalidades

- Ver todos los países en la página principal.
- Buscar países por nombre mediante un campo de texto.
- Filtrar países por región.
- Ver detalles completos de cada país en una página separada.
- Navegar entre países fronterizos desde la vista de detalle.
- Alternar entre **modo claro** y **modo oscuro**.
- Hero con imagen de mundo a pantalla completa.
- Infinite scroll estilo Instagram para cargar países progresivamente.

## 🚀 Tecnologías utilizadas

- [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [TailwindCSS](https://tailwindcss.com/)
- [React Router](https://reactrouter.com/)
- API [REST Countries](https://restcountries.com/)
